# digital-parliament-ui

User interface layer for the GBSB4U Digital Parliament.

## Purpose
This repository implements the visual and interactive layer
on top of the Digital Parliament Core v1.0.

The UI enables:
- Structured public debate
- Transparent proposal tracking
- Role-based access and views
- AI-assisted insights (non-authoritative)

## Core Principles
- Constitution-first design
- Human authority, AI assistance only
- Transparency by default
- Accountability at every step

## Scope (v1.0)
- Debate Rooms (read-only + moderated input)
- Proposal Viewer (status, history, rationale)
- Decision Trace View
- AI Insight Panels (explanatory, not prescriptive)

## Dependencies
- digital-parliament-core (v1.0)
